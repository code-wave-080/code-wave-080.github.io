## 문제
정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

## 풀이
### Javascript
```js
function solution(n) {
    let answer = 0;
  
    for (let i = 1; i <= Math.sqrt(n); i++) {
        if (n % i === 0) {
            answer += i + (i !== n / i ? n / i : 0);
        }
    }
    return answer;
}
```
